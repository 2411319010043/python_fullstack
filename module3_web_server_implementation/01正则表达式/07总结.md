正则表达式
    re.match(r'正则表达式',需要处理的字符串)
    1.匹配单个字符
        \d 0-9
        \w 大小写字符 0-9 中文日文 _
        \s 空格 tab
        ,[具体的筛选条件]，
        .能够匹配所有的字符除了\n re.S可以解决不匹配\n
        {} 限制前一个出现几次
        + 一个或多个,* 可以没有，一个或多个,? 可以有一个也可以没有
    2.re的高级用法 search 不会从头匹配
        sub 替换
        findall 查找匹配的结果
        split 切割
