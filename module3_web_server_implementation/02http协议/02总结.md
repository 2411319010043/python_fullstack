HTTP协议是基于tcp的超文本传输协议 是稳定的 是浏览器和服务器通信的协议 是一种规定格式的字符串

浏览器请求服务器：HTTP /1.1 200 OK 是必备的 是request

服务器回复浏览器：HTTP /1.1 200 OK 和 body 是必备的 是response 那么 请求头和请求体之间要有一个空行 这样浏览器就能够区分到底哪里是请求头哪里是请求体

再编写一个.py文件作为服务器的时候 要将换行写成 '\r\n'
    流程大致如下：
        1.创建TCP套接字
        2.绑定端口
        3.切换为监听模式
        4..accept(128)堵塞等待客户端来请求 
        5.创建新的套接字专门服务为某个客户端
        6..recv(1024)接收数据
        7.编写请求头和请求体 用.send()发送 要记得.encode('utf-8')
        8.关闭新套接字
        9.关闭监听套接字
