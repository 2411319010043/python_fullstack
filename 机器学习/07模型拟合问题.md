# 目标

1. 知道拟合是什么？
2. 理解过拟合、欠拟合是什么？
3. 知道过拟合、欠拟合出现的原因
4. 理解泛化是什么？

# 拟合

​	模型在训练集和测试集上的表现情况

## 欠拟合

​	模型在训练集和测试集上表现都不好

* 产生的原因：**模型过于简单**

## 过拟合

​	模型在训练集上表现好，在测试集上表现不好

* 产生的原因：**模型过于复杂**，数据不纯，训练数据太少

## 正好拟合

​	模型在训练集和测试集上表现都好

# 泛化

​	模型在**测试集**上的**表现好坏能力**，**模型的拟合情况=泛化能力**

## 奥卡姆剃刀原则

​	假设多个模型泛化能力是一样的，**较简单的模型**比较复杂的模型更可取
