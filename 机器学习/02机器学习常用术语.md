# 目标：

1. 知道样本是什么？

2. 知道特征是什么？

3. 知道标签/目标值是什么？

4. 理解数据集划分的方法

# 样本

​	**一行数据**就是一个**样本**，有时被叫成**一条记录**，**多个样本**组成**数据集**

# 特征

​	**一列数据**是**一个特征**，有时也被称作**属性**

# 标签/目标

​	模型要**预测**的那**一列数据**

# 数据集

​	**多个样本**组成**数据集**。
​	数据集分为：**训练集，测试集** 比例：**8：2，7：3**
​				 训练集*(train)*：用来训练模型*(model)*的数据集
​				 测试集*(test)*：用来测试模型的数据集
