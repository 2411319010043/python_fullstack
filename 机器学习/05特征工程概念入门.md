# 特征工程

​	利用专业背景知识和技巧**<u>处理数据</u>**，让机器学习算法效果更好。这个过程称为特征工程
​	**数据和特征**决定了机器学习的**上限**，而**模型和算法**只是**逼近**这个上限而已

1. 特征提取

   * 从原始数据中提取与任务相关的特征，构成特征向量。(会改变原数据)

2. 特征预处理 ——> 归一化、标准化(掌握)

   * 对特征进行预处理；防止因**量纲(单位)问题**，对模型产生影响。

     - 归一化 ——>(当前值- 最小值) / (最大值 - 最小值)

     - 标准化

3. 特征降维 

   * 将原始数据的维度降低，叫做特征降维，一般会对原始数据产生影响(例: 3维——>2维)

4. 特征选取

   * 原始数据特征很多，找出与任务相关的其中一个特征集合，不会改变原数据

5. 特征组合

   * 把多个特征合并成一个特征，利用乘法或加法完成
