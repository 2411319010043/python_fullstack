# 挂载

## what

mount就是‘挂载’，挂载就是把本地文件夹变成网站可访问的路径

## how

1. 准备文件夹

2. 导入StaticFiles

3. 创建处理器

4. 使用app.mount(挂载路径(字符串), StaticFiles(directory=‘’), name = ‘’)

示例

```python
from fastapi import FastAPI
from fastapi.staticfiles import StaticFiles
import uvicorn

app = FastAPI()

app.mount('/web', StaticFiles(directory='web/dist'), name='web')

if __name == '__main__':
    uvicorn.run(app)
```

## why

为了托管前端文件，实现前后端分离

# 重定向响应对象

经常与挂载一起使用

## what

告诉浏览器：跳转到另一个网址

>```
>用户在浏览器输入：http://localhost:8000/
>    ↓
>浏览器自动跳转到：http://localhost:8000/web/index.html
>    ↓
>显示实际的前端页面
>```
>

## how

* RedirectResponse(‘要跳转的路径’(必填))

```python
from fastapi.responses import RedirectResponse
from fastapi import FastAPI

app.mount('/web', StaticFiles(directory='web/dist'))

@app.get("/")
def home():
    return RedirectResponse("/web/index.html")  # 自动跳转

# 访问方式：
# http://localhost:8000/                ✅ 自动跳转到前端
# http://localhost:8000/web/index.html  ✅ 也能直接访问
```

## why

>```
># 没有重定向：用户需要记住复杂路径
># 用户必须输入：http://myapp.com/web/index.html
>
># 有重定向：用户只需输入简单域名
># 用户输入：http://myapp.com/
># 自动跳转到：http://myapp.com/web/index.html
>```
